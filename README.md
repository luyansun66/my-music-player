# 360° Immersive Audio Player

一个沉浸式暗黑风格3D音乐播放器，支持专辑轮播、动态背景氛围灯、音频频谱可视化。

## ✨ 功能特性

- **3D Coverflow 专辑轮播** — 基于 CSS 3D Transforms 的立体环绕专辑展示
- **动态背景氛围灯** — 实时音频频谱分析驱动，根据音乐低频/中高频能量动态调整光晕亮度和缩放
- **音频频谱可视化** — 底部 5 条频谱柱状图，分别对应低频(鼓点)、中低频(贝斯)、中频(人声)、中高频(乐器)、高频(镲片)
- **待播放队列系统** — 支持 5 首主列表 + 无限队列，歌曲播完自动从队列补充
- **导入歌曲** — 支持上传本地音频文件(.mp3)和封面图片，自动提取封面主色调生成氛围灯光
- **全屏模式** — 支持浏览器原生全屏 API
- **键盘快捷键** — 空格播放/暂停、左右键切换歌曲、上下键调节音量

## 🚀 使用方式

直接打开 `index.html` 即可运行，无需安装任何依赖。

```
open index.html
```

### 导入歌曲
1. 点击右下角 **"+"** 按钮
2. 填写歌曲名和歌手
3. 上传 .mp3 音频文件和封面图片
4. 点击确认导入，歌曲进入待播放队列
5. 点击队列列表中的歌曲即可播放

### 按钮位置（右下角）
```
📋（队列）   +（添加）   ⛶（全屏）
```

## 🛠 技术栈

- 原生 HTML + CSS + JavaScript（无框架依赖）
- Tailwind CSS（CDN）
- Web Audio API（实时频谱分析）
- CSS 3D Transforms（专辑轮播）
- CSS backdrop-filter（毛玻璃效果）
- Fullscreen API（全屏模式）

## 🎨 设计亮点

- **深炭黑背景** + 极细腻微噪点层（opacity 0.02），呈现天鹅绒质感
- **3D 环绕布局** — perspective 1800px，卡片环绕半径自适应
- **动态环境背光** — 基于 Web Audio API 实时分析音频频谱，驱动光晕呼吸脉动
- **快起慢落包络** — 攻击 0.25 / 释放 0.02-0.035，灯光跟随鼓点呼吸
- **毛玻璃 UI** — 所有交互按钮采用 backdrop-filter 毛玻璃效果
